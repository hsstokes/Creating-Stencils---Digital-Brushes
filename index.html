5. What's the correct order of steps to create a stencil?
Select white areas → Apply threshold → Delete selection
Apply threshold → Select white areas → Delete selection

Put the Steps in Order

Apply Threshold to image
Select white areas with Rectangular Marquee + Select Similar
Delete selected white areas
Position stencil on target image
Make selection and cut stencil shape into target
